Professional History

Intuit: Software Developer Co-op (January 2018 - December 2018)

I worked as a quality and tooling engineer to build tools that could be reused across multiple teams, in an effort to increase developer productivity. I played a pivotal role in automating a large scale CSS refactoring initiative across all of QuickBooks Online. I also worked on dashboards for quality metrics for web performance and end-to-end testing results. I got hands on experience with monitoring and maintaining a large scale microservices oriented architecture.

Additionally, I was also responsible for creating Cyphfell, an open source tool that automatically converts tests from the WebdriverIO framework for Selenium, and transforms them into tests that work with the Cypress framework.
